# Branding Assets

BeeLine loads a logo from this folder in one consistent app header location.

- `nolato_logo.png` is the approved Nolato logo asset for this app/repo.
- `nolato_logo_placeholder.png` is retained as a fallback placeholder.
- Set `BEELINE_LOGO_PATH` for a local override without changing committed assets.
- If no logo is present, the app falls back to text branding.
